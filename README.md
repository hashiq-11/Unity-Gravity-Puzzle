# Gravity Manipulation Puzzle

## How to Play
- **Movement:** WASD
- **Jump:** Space
- **Gravity Select:** Arrow Keys
- **Confirm Gravity:** Enter

## Deliverables
- **Windows Build:** Located in `/Builds/Hashiq_GravityPuzzle_Windows_Build.zip`
- **macOS Build:** Due to a Windows-only development environment, a native macOS export was not possible. However, the codebase is platform-agnostic and ready for a macOS build via the Unity Editor.
